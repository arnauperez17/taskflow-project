## Conectar servidores MCP

Primero investigué qué es MCP (Model Context Protocol). Es un protocolo que permite a la IA acceder a información del proyecto, como archivos o repositorios.

Después configuré MCP en Cursor creando el archivo mcp.json y añadiendo un servidor filesystem.

Este servidor permite que la IA lea los archivos del proyecto.

Después probé varias consultas en el chat de Cursor:

1. Muestra los archivos de este proyecto
2. Explica qué hace el archivo app.js
3. Qué funciones hay en este proyecto
4. Explica cómo funciona la búsqueda de tareas
5. Describe la estructura del proyecto

La IA pudo leer el código y responder usando información del proyecto.

En proyectos reales MCP puede ser útil para que la IA entienda mejor el contexto del código, analice archivos automáticamente y ayude a los desarrolladores a trabajar más rápido.
